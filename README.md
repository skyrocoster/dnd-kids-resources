🎲 Spell Cards • 🛡️ Condition Cards • ⚔️ Weapons • 🎯 Skills

## Quick Start

**Option 1: Just Open the Browser**

Open `index.html` directly in your browser. Weapons, Character Sheet, and Trackers work immediately offline.

**Option 2: Run with Database Features** (Spells, Conditions, Creatures, Skills)

```bash
python server_flask.py
# Then visit http://localhost:8000
```

**Need setup instructions?** See [Getting Started Guide](docs/guides/GETTING_STARTED.md)

## What Is This?

A web-based toolkit creating **printable D&D reference cards** (63.5mm × 88.9mm—standard playing card size).

**Features:**
- 54 Spell Cards (filterable by level)
- 19 Condition Cards
- 42+ Weapon Cards
- 18 Skill Cards
- 6 Creature Cards
- Character Sheet (printable)
- HP Tracker (interactive)
- Turn Order Tracker
- **Black & White print preview** (save ink!)
- **Cards print 9 per A4 page**
- Database-driven content (SQLite + Flask API)
- AI-powered spell/creature parser
