# Analysis Archive

Historical research and accepted decision rationale live here. These documents explain why earlier work took a
particular direction, but they do not define current behavior; canonical references and area guides do.

## Structure

- `Decisions/` contains accepted architecture, schema, or product decisions.
- This directory contains completed investigations and source-data audits.

## Use

Read an archived analysis or decision only when tracing contract provenance. Update it only to repair archival
metadata or links. Create a new analysis or decision when its rationale will remain useful after implementation;
put current behavior in the appropriate canonical reference or area guide instead.
