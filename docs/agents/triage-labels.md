# Triage Label Vocabulary

The five canonical triage roles use their names as labels:

| Label | Meaning |
|-------|---------|
| `needs-triage` | Issue needs initial triage (not reviewed yet) |
| `needs-info` | Issue needs more information from author before proceeding |
| `ready-for-agent` | Issue is clear and ready for AI/agent work |
| `ready-for-human` | Issue is clear and ready for human review or action |
| `wontfix` | Issue will not be addressed |

## Consumer

The `triage` skill reads and applies these labels. If you rename them, update this table and re-run the setup.
