# Issue Tracker — D&D Kids Resources

Issues live in [GitHub Issues](https://github.com/skyrocoster/dnd-kids-resources/issues).

## How to create an issue

Use the `gh` CLI:

```bash
gh issue create --title "..." --body "..."
```

Or navigate to [Issues](https://github.com/skyrocoster/dnd-kids-resources/issues) and click **New issue**.

## How skills use this

- `to-tickets`: converts sketches and problem statements into GitHub issues
- `triage`: reads and applies labels to issues
- `qa`: uses issues to track test cases and coverage
- `to-spec`: converts issues into specification documents

## Notes

- Pull requests are not surfaced to the triage queue (external PRs can be enabled in `triage-labels.md` if needed later)
- Issue labels are configured in `docs/agents/triage-labels.md`
